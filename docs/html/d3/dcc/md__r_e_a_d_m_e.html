Introducción

Este proyecto consiste en una aplicación de venta de dulces online con un carrito funcional.

En la web puedes:

  • Entrar como invitado, pero sin poder realizar ningún pedido.
  • Registrarte/Iniciar sesión, con acceso a realizar pedidos y eliminarlos.
  • Entrar como administrador para añadir, modificar o eliminar productos.

Requerimientos

  • PHP 7.0 o superior
  • MySQL
  • Composer 2.5.1

Tener un gestor como XAMPP para mantener la base de datos activa localmente.

Instalación

  1. Descargue o clone el repositorio en su servidor, para clonarlo utiliza este comando:
{shell=}
git clone https://github.com/Javimartel/Online-Shop-PHP.git
  1. La aplicación utiliza variables de entorno por lo que para utilizar la base de datos hay que crear un fichero llamado ".env" en el directo raíz de la siguiente manera:
{shell=}
DB_HOST=''
DB_USER=''
DB_PASS=''
DB_NAME='tienda'

Rellenando los campos vacíos con los datos que vaya a utilizar localmente.

  1. Lo siguiente será instalar las dependencias necesarias para el funcionamiento de la aplicación utilizando "*composer*" con este comando:
{shell=}
composer install
  1. Ahora tiene que ir a XAMPP y activar el servicio de MySQL:

  1. Debe iniciar el servidor de PHP en el archivo *"installer.php"* para que se instale la base de datos con sus respectivos datos; esta es la ruta:
    {shell=}
    ./config/installer.php
    Debería aparecer este mensaje:

  1. Ahora simplemente debe ejecutar de nuevo el servidor en el *"index.php"*:
{shell=}
./index.php

O acceder mediante localhost:

{shell=}
localhost:3000/index.php

Listo, ya puede disfrutar de la aplicación.
